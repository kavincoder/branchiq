# BranchIQ Roadmap

Planned features and improvements for future releases.

## v1.1 — Alerts & Notifications
- [ ] Email alerts for flagged anomalies (SendGrid integration)
- [ ] SMS notifications for high-risk transactions (Twilio)
- [ ] In-app notification bell with real-time updates (WebSocket)

## v1.2 — Multi-Branch Support
- [ ] Branch hierarchy — regional manager role with cross-branch visibility
- [ ] Per-branch analytics and comparison dashboard
- [ ] Branch-level export with consolidated reports

## v1.3 — Advanced Analytics
- [ ] Customer 360 view — full history per account number
- [ ] Loan repayment forecast chart (amortization schedule)
- [ ] Monthly P&L summary export (PDF)

## v2.0 — Mobile App
- [ ] React Native companion app for branch managers
- [ ] Push notifications for anomaly alerts
- [ ] Offline mode with sync on reconnect

## Ongoing
- [ ] Expand ML model — retrain monthly on new data automatically
- [ ] Add more anomaly reasons (velocity checks, geo-anomalies)
- [ ] OpenAPI spec auto-generated client (TypeScript SDK)
